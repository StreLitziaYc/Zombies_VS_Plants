## 项目背景
2023-2024学年秋季学期SCU研究与开发课程作业
## 简介
复现了植物大战僵尸中，放置僵尸吃植物的小游戏  
总共有三个关卡、三种僵尸、五种植物
## 环境要求
使用python的pgzero库进行开发  
`pip install pgzero`
